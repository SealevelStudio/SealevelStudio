# Logo Files

## Transaction Builder Logo
- **File**: `transaction-builder-logo.jpeg`
- **Location**: `/public/transaction-builder-logo.jpeg`
- **Usage**: Background logo in the transaction builder drop zone
- **Recommended specs**:
  - Format: JPEG or PNG
  - Size: 400-800px width (maintain aspect ratio)
  - Opacity: Will be displayed at 8% opacity
  - Background: Dark background works best

## Landing Page Logo
- **File**: `logo-video.mp4` (video format)
- **Location**: `/public/logo-video.mp4`
- **Usage**: Animated logo on landing page

## Other Logos
- `sea-level-logo.png`: General Sea Level Studio logo
- `apple-touch-icon.png`: iOS app icon

---

**Note**: Place your transaction builder logo file at:
`/public/transaction-builder-logo.jpeg` (or `.png`)

The logo will automatically appear in the transaction builder once the file is added!
